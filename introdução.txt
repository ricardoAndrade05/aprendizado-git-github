Git é um sistema de controle de versão de arquivos. Com ele podemos desenvolver projetos colaborativos com facilidade. 
-> Registra diferentes modificações de um arquivos
-> Permite recuperar uma versão específica
-> Pode resolver os seguintes problemas:
  * Perda de arquivos importantes.
  * Resgate de versão anterior.

GitHub é um serviço para hospedar projetos Git. Permite projetos privados ou open source. Ajuda a desenvolver um networking entre programadores e serve como um portfólio online.
